
# 🧱 Brick Breaker

A modern Brick Breaker game built with **React + Vite**.
Destroy all the bricks, keep the ball in play, and beat your high score!

## 🎮 Features

* 🕹️ Classic brick breaker gameplay
* 🎯 Persistent Score tracking hosted on Firebase Firestore
* 🧱 Brick collision detection
* 🏓 Paddle movement controls
* 💥 Ball physics & bounce logic
* ❤️ Lives system
* 🔁 Game reset functionality
* ⚡ Built with Vite for fast development

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/OlivierMarteaux/r002_first_brick_breaker.git
cd r002_first_brick_breaker
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The app will run at:

```
http://localhost:3000
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🎯 How to Play

* Use **Left / Right Arrow Keys** or mouse, or finger on touch device to move the paddle.
* Bounce the ball to break all bricks.
* Don’t let the ball fall below the paddle.
* Clear all bricks to win!
